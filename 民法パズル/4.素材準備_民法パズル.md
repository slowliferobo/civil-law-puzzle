# 4. 素材準備リスト: 民法パズル

## 1. グラフィック素材
### UI素材
*   [ ] **ブロック画像**: 用語用（赤系）、定義用（青系）のベース画像（CSSで代用可だが、リッチにするなら画像）
*   [ ] **背景画像**: 六法全書や法廷をイメージした落ち着いた背景
*   [ ] **ボタン**: スタート、リトライボタンのデザイン
*   [ ] **アイコン**: 正解時の「○」、不正解時の「×」
*   [ ] **フォント**: 読みやすいゴシック体（Google Fonts: Noto Sans JPを使用中）

---

## 2. サウンド素材
### BGM
*   [ ] **メインテーマ**: 集中できる静かな曲（ローファイ・ヒップホップやアンビエント系）
*   [ ] **リザルト画面**: 少し明るめのジングル

### SE（効果音）
*   [ ] **決定音**: ボタンクリック時
*   [ ] **ドラッグ開始音**: ブロックを持ち上げた時
*   [ ] **正解音**: ブロックが消える時（爽快感のある音）
*   [ ] **不正解音**: ブロックが弾かれる時（ブブーッという音）
*   [ ] **クリア音**: 全て消した時

---

## 3. エフェクト素材
*   [ ] **消滅エフェクト**: ブロックが消える時のパーティクル
*   [ ] **コンボエフェクト**: 連続正解時のキラキラ演出

---

## 4. 素材調達計画
*   **グラフィック**: CSS装飾を基本とし、必要に応じてAI生成（generate_image）またはフリー素材を使用。
*   **サウンド**: フリー素材サイト（魔王魂、効果音ラボなど）から選定、またはAI生成ツールが使えるなら使用。
*   **実装方針**: まずはプレースホルダー（CSSのみ）で進め、最後に差し替える。
