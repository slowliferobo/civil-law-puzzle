# トラブルシューティング・テンプレート

ゲームが動かない、表示がおかしい等の問題が発生した場合のチェックリストです。

## 1. 画面が真っ白 / 何も表示されない

- [ ] **コンソールエラーの確認** (PC: F12キー -> Consoleタブ)
    - 赤いエラーが出ていませんか？
    - `Uncaught TypeError: Cannot read properties of null` -> HTML要素が見つかっていません。IDが正しいか、JSの読み込み順を確認してください。
    - `SyntaxError` -> コードの書き間違い（閉じカッコ忘れなど）があります。
- [ ] **ファイル読み込みの確認** (Networkタブ)
    - `style.css` や `script.js` が 404 (Not Found) になっていませんか？
    - ファイル名の大文字・小文字は合っていますか？

## 2. レイアウト崩れ / 要素が表示されない

- [ ] **画面サイズ・向き**
    - PCの場合、ウィンドウ幅を狭くしたり広くしたりして変化しますか？
    - スマホの場合、横画面・縦画面で意図した表示になりますか？
- [ ] **要素の重なり (z-index)**
    - 見えない要素が他の要素の下に隠れていませんか？
- [ ] **タイミング問題 (Race Condition)**
    - JSでサイズ計算をしている場合、描画完了前に計算して `0` になっていませんか？
    - **対策**: `setTimeout` や `requestAnimationFrame` で処理を少し遅らせてみてください。

## 3. ボタンや操作が効かない

- [ ] **イベントリスナー**
    - `addEventListener` は正しく設定されていますか？
    - エラーでJSが途中で止まっていませんか？
- [ ] **キャッシュ (Cache)**
    - ブラウザが古いファイルを記憶していませんか？
    - **対策**: スーパーリロード (Ctrl+F5) やシークレットウィンドウで試してください。
- [ ] **前面の要素**
    - 透明な要素がボタンの上に被さってクリックを邪魔していませんか？

## 4. 公開・デプロイ (GitHub Pages)

- [ ] **404 エラー**
    - リポジトリ設定の Pages で `Source` が正しく設定されていますか？ (`main` / `root`)
    - `index.html` はルートフォルダにありますか？
- [ ] **更新が反映されない**
    - Pushしてから数分待ちましたか？
    - ブラウザのキャッシュをクリアしましたか？

---
**AIへの報告用フォーマット**

問題が解決しない場合、以下の情報をAIに伝えてください：

1. **起きている現象**: （例：スタートボタンを押しても反応がない）
2. **環境**: （例：iPhoneのSafari、WindowsのChrome）
3. **コンソールエラー**: （あればエラーメッセージをコピペ）
4. **試したこと**: （例：リロードしたが直らない）
