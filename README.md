# 民法マスター：条文パズル (v2.0)

## 概要
民法の用語と定義をマッチングさせるパズルゲームです。
「桜井政博メソッド」に基づき、リスクとリターン、爽快な演出、スマホ対応（PWA）が実装されています。

## デプロイ方法（Web公開）

このフォルダの中身をそのままWebサーバーにアップロードするだけで動作します。

### GitHub Pages を使う場合（推奨）
1.  GitHubに新しいリポジトリを作成します。
2.  このフォルダ内のファイル（`index.html`, `script.js`, `style.css`, `manifest.json`）をアップロードします。
3.  リポジトリの `Settings` -> `Pages` に移動します。
4.  `Source` を `Deploy from a branch` にし、`Branch` を `main` (または `master`) に設定して `Save` します。
5.  数分後、発行されたURLにアクセスすれば遊べます！

### Netlify / Vercel を使う場合
1.  Netlify または Vercel にログインします。
2.  このフォルダをドラッグ＆ドロップするだけで公開されます。

## スマホでの遊び方
1.  公開されたURLをスマホ（Safari/Chrome）で開きます。
2.  「共有」ボタン -> 「ホーム画面に追加」を選択します。
3.  ホーム画面に追加されたアイコンをタップすると、アプリのように全画面で遊べます。
