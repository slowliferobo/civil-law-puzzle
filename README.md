# 民法マスター：条文パズル

**民法の条文をパズルで覚える学習ゲーム**

*   **概要**: 用語と定義をマッチングさせ、ゲーム感覚で条文知識を定着させます。公務員・資格試験受験生や法学部生の隙間学習に最適。
*   **特徴**: 直感的なドラッグ＆ドロップ操作、タイムアタック、スコア評価機能。
*   **使い方**: インストール不要。`index.html`（または`民法パズルを起動.bat`）を開くだけでブラウザですぐに遊べます。
*   **状態**: 開発中プロトタイプ。今後、問題数や機能を拡充予定です。
*   **サポート**: バグ報告や要望は[GitHub Issues](https://github.com/slowliferobo/civil-law-puzzle/issues)までお願いします。
