# 民法マスター：条文パズル (テスト版)

この度はテストプレイにご協力いただきありがとうございます。

## 遊び方
1. `index.html` をダブルクリックして、ブラウザ（Chrome, Edge, Safariなど）で開いてください。
2. タイトル画面の「スタート」ボタンを押すとゲームが始まります。
3. 画面上の「用語（赤）」と「定義（青）」のブロックを組み合わせてください。
    - **ドラッグ＆ドロップ**: ブロックを掴んで重ねる
    - **クリック**: 片方をクリックして選択し、もう片方をクリックしてマッチング
4. 制限時間内に全てのペアを消すとクリアです。
